## Starting Up Project
To start up backend (FastAPI): 
  - Start up venv 
  - Run --> 'pip install -r requirements.txt'
  - Run --> 'uvicorn main:app --reload'

To start up frontend (Vite (react)): 
  - 'npm install'
  - 'npm run dev'

Backend: http://localhost:8000 (API endpoint)

Frontend: http://localhost:5173 (React app)
